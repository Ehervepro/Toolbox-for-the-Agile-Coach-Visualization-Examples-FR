 Matrice de Pair Programming
 
Est ce que vous essayez de faire plus de Pair Programming au sein de votre équipe? Aimeriez-vous visualiser le temps que vous y passer au cours d'une semaine? Essayez de mettre en place une matrice de Pair Programming.
 
Chaque fois que vous avez fait une séance de Pair Programming, faîtes une croix dans la case correspondante. Vous devez vous accordez en tant qu'équipe sur le temps de binômage nécéssaire correspondant à une croix.

Fixer une date pour faire une revue et discuter des résultats. Si vous souhaitez continuer le suivi de votre Pair Programming, réinitialiser la matrice et convenez d'une nouvelle date de revue.

 Pair Programming Matrix

Are you trying to do more pair programming in your team? Would you like to visualize how much you pair during a week? Try putting up a Pair Programming Matrix. 

Whenever you’ve done a pairing session, make a tick in the corresponding box. You need to agree as a team how much pairing is required for a tick.

Set a date when you review and talk about the results. If you want to continue tracking your pairing, reset the matrix and agree upon a new review date.
