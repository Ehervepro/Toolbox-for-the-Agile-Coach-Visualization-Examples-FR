Une légende selon le type de User Story 

Certaines équipes trouvent pratique de distinguer les différents types de user stories. Tous les sujets priorisés ne rentrent pas dans le format typique orienté client d'une User Story.

Plutôt que de vous forcer à écrire une carte étrange au format User Story pour un correctif de release, une investigation ou expérience timeboxée, autorisez des formats plus simples pour les user stories non centrées utilisateurs. Rendez les différentes variétés de User Stories claires en utilisant des couleurs et ajouter ces couleurs sur la légende de votre mur.

User Story Flavor Legend

Some teams find it useful to distinguish between different types of user stories. Not all prioritized work fit the typical customer value focused User Story template. 

Instead of forcing yourselves to write a weird card using User Story format for a patch release or a time boxed investigation or experiment, allow simpler templates when working with non-user centric stories. Make the different flavors of User Stories clear by the use of colors and add the colors to your wall legend.
